Detalhamento de Processo - Paralelismo

Paralelismo

Obrigatóriamente os próximos procedimentos são independentes entre si, mas acontecém simultaneamente.

Procedimentos Paralelos(Não importa quantos sejam os procedimentos.

# Procedimento Descrição
1 Paralelo A Desenvolvor atividade A
2 Paralelo B Desenvolvor atividade B
3 Paralelo C Desenvolvor atividade C

Múltiplas Escolhas
